# OA's Node Helper

一些常用的小工具 🤝


## 說明
很簡單就是一個 OA 個人 [Node.js](https://nodejs.org/) 的 [npm](https://www.npmjs.com/) 小幫手！

## 安裝

```shell
npm install @oawu/helper
```

## 使用

引入 `require('@oawu/helper')` 即可使用，如下範例：

```javascript

  const Helper = require('@oawu/helper')
  Helper.println('Hi~')

```
